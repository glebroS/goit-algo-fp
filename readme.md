# Фінальний проєкт: Алгоритми та структури даних

## Опис завдань

### Завдання 1: Структури даних (Однозв'язний список)
Файл: `task_1.py`
Реалізовано однозв'язний список з методами:
- Реверсування списку
- Сортування злиттям (Merge Sort)
- Об'єднання двох відсортованих списків

### Завдання 2: Рекурсія (Дерево Піфагора)
Файл: `task_2.py`
Програма візуалізує фрактал "Дерево Піфагора" за допомогою бібліотеки `turtle`. Користувач може задати рівень рекурсії.

### Завдання 3: Алгоритм Дейкстри
Файл: `task_3.py`
Реалізація алгоритму Дейкстри для знаходження найкоротшого шляху у графі з використанням бінарної купи (`heapq`).

### Завдання 4: Візуалізація бінарної купи
Файл: `task_4.py`
Побудова та візуалізація бінарної купи як дерева за допомогою бібліотек `networkx` та `matplotlib`.

### Завдання 5: Візуалізація обходу дерева
Файл: `task_5.py`
Візуалізація обходів дерева в глибину (DFS) та в ширину (BFS). Кольори вузлів змінюються від темних до світлих відтінків залежно від порядку обходу.

### Завдання 6: Жадібні алгоритми та динамічне програмування
Файл: `task_6.py`
Розв'язання задачі вибору їжі з найбільшою калорійністю в межах бюджету двома способами:
1. Жадібний алгоритм (максимізація співвідношення калорії/вартість).
2. Динамічне програмування (знаходження оптимального рішення).

### Завдання 7: Метод Монте-Карло
Файл: `task_7.py`
Симуляція кидків двох кубиків для обчислення ймовірностей сум чисел.

#### Висновки до завдання 7
Метод Монте-Карло використовувався для імітації 1,000,000 кидків двох гральних кубиків. Отримані емпіричні ймовірності порівнювалися з теоретичними (аналітичними) розрахунками.

| Сума | Ймовірність (Монте-Карло) | Ймовірність (Аналітична) | Відхилення |
|------|---------------------------|--------------------------|------------|
| 2    | 2.79%                     | 2.78%                    | 0.01%      |
| 3    | 5.53%                     | 5.56%                    | 0.03%      |
| 4    | 8.35%                     | 8.33%                    | 0.02%      |
| 5    | 11.06%                    | 11.11%                   | 0.05%      |
| 6    | 13.87%                    | 13.89%                   | 0.02%      |
| 7    | 16.68%                    | 16.67%                   | 0.01%      |
| 8    | 13.91%                    | 13.89%                   | 0.02%      |
| 9    | 11.13%                    | 11.11%                   | 0.02%      |
| 10   | 8.34%                     | 8.33%                    | 0.01%      |
| 11   | 5.54%                     | 5.56%                    | 0.02%      |
| 12   | 2.80%                     | 2.78%                    | 0.02%      |

**Висновок:** Результати симуляції методом Монте-Карло дуже близькі до аналітичних значень. Відхилення мінімальні (в межах похибки випадковості), що підтверджує правильність роботи алгоритму симуляції та закону великих чисел. Метод Монте-Карло дозволяє з високою точністю наближати теоретичні ймовірності при достатньо великій кількості експериментів.
